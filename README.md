# SumSynth-Web
Interactive WebPage for SumSynth

## Start Server

### Install Backend tool ``SumSynth``

```sh
$ cd server && ./install.sh
```


### Install Web Server Dependencies
Server currently uses **Node version 17**

```sh
$ npm i
$ npm i -g nodemon
```

### Run Server

```sh
$ nodemon index.js
```


## Start Client
```sh
$ cd sumsynth-client
$ npm start
```
